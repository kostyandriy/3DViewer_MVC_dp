Инструкция по использованию 3D_Viewer_v2:

Установка приложения:

1. При помощи терминала зайдите в папку src

2. Введите команду make

Запуск приложения:

1. Перейдите в папку build

2. Запустите приложение, выполнив команду "./3D_Viewer_2_0" или "open 3D_Viewer_2_0.app".

Загрузка модели:

1. В главном окне введите нажмите на кнопку "Open file" и выберите объектный файл - загрузится каркасная модель.

Взаимодействие с моделью:

1. Перемещение: перемещать модель можно при помощи ползунков под "Translate Model"; по осям x,y,z соответственно

2. Поворот: поворачивать модель можно при помощи ползунков под "Rotate Model"; по осям x,y,z соответственно

3. Масштаб: можно увеличивать или уменьшать размер модели при помощи ползунка под "Scale Model"; по осям x,y,z соответственно

Настройки:

1. Тип проекции: можно настраивать тип проекции при помощи соответствующих кнопок

2. Рёбра: можно настроитть тип ребер, их толщину и цвет.

3. Вершины: можно настроить тип отображения вершин, величину отображаемых вершин и их цвет в рамке;

4. Фон: можно выбрать цвет фона по кнопке "Set Background";

5. Все выбранные настройки сохраняются между перезапусками программы.

Сохранение изображения:

1. Загрузите каркасную модель

2. Нажмите кнопку "Screenshot", выберите формат скриншота и дирректорию и нажмите save - в директории сохранится скриншот модели

Создание скринкаста:

1. Загрузите каркасную модель

2. Для создания скринкаста текущих пользовательских преобразований модели в GIF-анимацию с размером 640x480, 10 кадрами в секунду и продолжительностью 5 секунд, нажмите кнопку GIF

3. Выберите название гифки, директорию сохранения и нажмите save

4. Вертите моделью как вздумается

5. GIF-анимация будет сохранена в выбранной директории
